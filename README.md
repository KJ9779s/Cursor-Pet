# 9779s Browser Pet

這是一個為 9779s 設計的滑鼠精靈擴充功能。
A browser extension designed for 9779s , featuring interactive Browser Pet.

## 功能 / Features
1. **角色互動**：包含 KOKO、kaitom、kaitoon、kaotung 等多個角色。 (Includes multiple characters like KOKO, kaitom, kaitoon, and kaotung.)
2. **滑鼠追蹤**：角色會自動追隨你的滑鼠游標。 (Characters automatically follow your mouse cursor.)
3. **自動模式**：當滑鼠長時間閒置時，角色會自動切換狀態。 (Automatically switches modes when the mouse is idle.)
4. **自訂設定**：可自由調整角色大小、移動速度以及游標跟隨距離。 (Customizable scale, speed, and follow distance.)

## 隱私權政策 / Privacy Policy
本擴充功能非常重視您的隱私。所有的設定參數（如角色類型、大小、速度）僅透過 `chrome.storage.local` 儲存於您的瀏覽器本地端。
This extension respects your privacy. All settings (e.g., character type, scale, speed) are stored locally in your browser using `chrome.storage.local`.

* **資料收集**：我們不會收集、儲存或傳輸任何個人資訊至外部伺服器。 (Data Collection: We do not collect, store, or transmit any personal data to external servers.)
